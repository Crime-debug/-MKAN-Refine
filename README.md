# MKAN-Refine

Official implementation of the paper:

**MKAN-Refine: Fine-Grained Crisis Information Mining via Reliability-Aware Nonlinear Refinement and Kolmogorov-Arnold Networks**

## Overview

This repository provides the inference code and core implementation for **MKAN-Refine**, a multimodal framework for fine-grained crisis information mining. The model combines CLIP-based multimodal representations with nonlinear refinement and reliability-aware fusion for humanitarian classification.
